# RetinexFormer: One-stage Retinex-based Transformer for Low-light Image Enhancement


This repository is a PyTorch implementation of the **RetinexFormer** architecture for low-light image enhancement, completed as a Major Project. The model leverages the Retinex theory (decomposing images into illumination and reflectance) and utilizes a Transformer-based "Illumination-Guided Attention" (IGA) mechanism.

## 🌟 Features
- **Theory-Driven:** Combines classic Retinex theory with modern Vision Transformers.
- **Efficient Inference:** Includes a patch-based inference script with smooth blending (Hann window) to handle high-resolution images.
- **Detailed Evaluation:** Trained and tested on the LOL (Low-Light) dataset.

---

## 🏗️ Architecture
RetinexFormer consists of an Illumination Estimator and a Lighting-Aware Transformer. It effectively captures global context to restore brightness while suppressing noise in the reflectance map.
